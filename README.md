## What is this and why?
A bash script that easilly adds a .exe file to your linux applications menu!
I made this because i wanted an easier way to add .exe files to the application menu 

## Installation
```
git clone https://github.com/ninetta12/exe-to-applicationmenu/
```
```
cd exe-to-applicationmenu
```

## Usage
```
sh exe-to-desktop.sh path_to_file.exe
```
